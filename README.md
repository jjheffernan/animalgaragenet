# Animal Garage

Automotive brand touchpoint — a merchandising-forward digital marketing site for Animal Garage. Shop drops, brand media, and garage culture in one place.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start dev server         |
| `npm run build`   | Production build         |
| `npm run preview` | Preview production build |
| `npm run check`   | Type-check               |
| `npm run lint`    | ESLint + Prettier check  |
