# Locale & i18n

International readiness is scaffolded but not fully wired to Saleor channels yet.

## Files

| File                                       | Purpose                                    |
| ------------------------------------------ | ------------------------------------------ |
| `src/lib/types/locale.ts`                  | Locale codes, configs, shipping regions    |
| `src/lib/i18n/locale.ts`                   | `getLocaleConfig()`, `isSupportedLocale()` |
| `src/lib/i18n/currency.ts`                 | `formatMoney()` via `Intl.NumberFormat`    |
| `src/lib/stores/locale.svelte.ts`          | Reactive locale state (Svelte 5 class)     |
| `src/lib/components/LocaleSelector.svelte` | UI picker                                  |

## Supported locales

| Code    | Label          | Currency | Regions  |
| ------- | -------------- | -------- | -------- |
| `en-US` | United States  | USD      | US, CA   |
| `en-GB` | United Kingdom | GBP      | GB, EU   |
| `de-DE` | Deutschland    | EUR      | DE, EU   |
| `ja-JP` | 日本           | JPY      | JP, APAC |

## Usage in components

```svelte
<script lang="ts">
	import { locale } from '$lib/stores/locale.svelte';
</script>

<p>{locale.formatPrice(product.pricing.priceRange.start.amount)}</p>
```

`formatPrice` uses the current locale's currency from `LocaleConfig`.

## Default locale

Set via environment:

```env
PUBLIC_DEFAULT_LOCALE=en-US
PUBLIC_DEFAULT_CURRENCY=USD
```

Read in `src/lib/config/env.ts`, applied in `locale.svelte.ts` on init.

## Changing locale programmatically

```typescript
import { locale } from '$lib/stores/locale.svelte';
locale.setLocale('de-DE');
```

LocaleSelector component calls this on user selection.

## Future: Saleor channel mapping

Each locale will map to a Saleor channel for pricing/inventory:

```
en-US → us-channel (USD)
en-GB → uk-channel (GBP)
de-DE → eu-channel (EUR)
ja-JP → jp-channel (JPY)
```

Pass channel to GraphQL queries:

```typescript
await saleorFetch(PRODUCTS_QUERY, { channel: getChannelForLocale(locale.code) });
```

## Adding a new locale

1. Add code to `LocaleCode` union in `locale.ts`
2. Add entry to `locales` array with currency and shipping regions
3. LocaleSelector auto-lists all entries
4. Map to Saleor channel when commerce is wired
